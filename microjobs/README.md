# MicroJobs

## Descrição
O **MicroJobs** é uma plataforma web que conecta freelancers a pequenos trabalhos (microtasks), permitindo que usuários publiquem tarefas e outros usuários realizem-nas em troca de pagamento. O objetivo do projeto é facilitar a realização de trabalhos rápidos de forma segura e organizada.

## Funcionalidades
- Cadastro e login de usuários
- Telas para o cliente e para o profissional
- Avaliações e feedback

## Tecnologias Utilizadas
- **Frontend:** HTML, CSS, JavaScript (depois vamos passa-lo para o React)
- **Backend:** (Python + Flask)
- **Mobile:** (React-Native + Node.js)
- **Banco de Dados:** (Postgres)
- **Autenticação:** JWT / OAuth
- **Outras ferramentas:** (Postman para testes)
